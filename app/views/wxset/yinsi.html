收集信息用途说明
{foreach $privacyArr as $k=>$item}
{if $k == 'UserInfo'}
例如 辨别用户身份
{elseif $k == 'Address'}
例如 地址导航
{elseif $k == 'PhoneNumber'}
例如 发送短信通知
{elseif $k == 'MessageFile'}
例如 上传评价图片等
{elseif $k == 'Location'}
例如 推荐最近的自提点
{elseif $k == 'Album'}
例如 评价晒单
{elseif $k == 'AlbumWriteOnly'}
例如 保存商品图片和海报
{elseif $k == 'ChooseLocation'}
例如 选择位置作为收货地址
{elseif $k == 'Clipboard'}
例如 复制订单号
{else}
选填
{/if}
{/foreach} 信息收集方联系方式
